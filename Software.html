FLL Timer application

Every year my robotics team (FRC team 3683 – Team Dave) hosts a First Lego League competition. I made this app to be used with the OBS streaming software to control a timer with video overlay on a projector. This app features a menu system, 3 display modes (practice match, match, and break). These feature different times on the timer and displays what mode it is in. this is the second variation of the app.

Modified Snake Game

I followed a tutorial to create a snake game in my early java days. After finishing the tutorial I modified the game to add speed to your snake as you meet score benchmarks and added sounds to the game to learn how to use sound in java.

Console X's and O's

This is a console app that I made as one of my first projects in java. I was challenged by my friend to create a X’s and O’s app in the console using scanners and printing. I was pretty proud to have finished this challenge within 2 hours, although now I could probably redo it in 30 minutes.
